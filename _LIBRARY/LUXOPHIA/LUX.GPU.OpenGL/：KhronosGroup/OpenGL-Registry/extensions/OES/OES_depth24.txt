Overview

    This extension enables 24-bit depth components as a valid
    render buffer storage format.

Issues

 
New Tokens

    Accepted by the <internalformat> parameter of RenderbufferStorageOES:

        DEPTH_COMPONENT24_OES                 0x81A6
